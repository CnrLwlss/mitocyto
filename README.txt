=========
mitocyto
=========

Image analysis software for detecting cells in serial sections and summarising intensity in each cell across multiple channels.  Includes automatic segementation and gui-based manual segmentation workflows.

Source code and documentation on GitHub: `https://github.com/CnrLwlss/mitocyto <https://github.com/CnrLwlss/mitocyto/>`_.
